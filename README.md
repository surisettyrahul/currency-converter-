# Currency Converter

A simple and stylish web-based currency converter that converts amounts from one currency to another using live exchange rates from ExchangeRate-API.

## Features

- Real-time currency conversion between over 160 currencies
- Dynamic currency dropdowns populated from up-to-date API data
- Clean and modern UI built with HTML, CSS
- Easy to use form input for amount and currency selection
- Responsive design for different screen sizes

## Technologies Used

- HTML5 for markup
- CSS3 for styling with CSS variables for theme colors
- Vanilla JavaScript for fetching API data and currency conversion logic
- ExchangeRate-API to provide live exchange rates in JSON format

## API

This project uses the free ExchangeRate-API endpoint:

https://api.exchangerate-api.com/v4/latest/USD
